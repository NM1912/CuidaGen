# 🧬 CuidaGen

O *CuidaGen* é um projeto de extensão voltado à educação em genética, promoção da saúde e divulgação científica.

## Objetivos

- Promover educação em saúde e genética.
- Desenvolver materiais educativos.
- Incentivar a divulgação científica.
- Aproximar a universidade da comunidade.

## Funcionalidades do site

- Página inicial
- Quem Somos
- Publicações
- Galeria
- Contato

## Tecnologias utilizadas

- HTML5
- CSS3
